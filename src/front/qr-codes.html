QR Codes pour Places de Parking

Instructions d'utilisation

Pour les administrateurs/secrétaires :

  • Cliquez sur "Imprimer tous les QR codes" pour générer une version imprimable
  • Découpez et plastifiez chaque QR code
  • Fixez chaque QR code à la place de parking correspondante

Pour les employés :

  • Scannez le QR code de votre place réservée avec votre smartphone
  • Ou entrez manuellement le numéro de place sur la page principale
  • Le check-in doit être effectué avant 11h00
